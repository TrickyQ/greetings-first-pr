# Greetings

A tiny practice project for shipping a first pull request.

## What it does

It provides a `greet()` function that returns a friendly greeting, and a
`farewell()` function that returns a friendly goodbye.

## Usage

```python
from greetings import greet, farewell

print(greet("Sam"))     # Hello, Sam!
print(farewell("Sam"))  # Goodbye, Sam. See you soon!
```

## Running the tests

```bash
python -m pytest
```
